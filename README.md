## ts 脚手架工具

拉取 vite + vue3 + pinia + vue-router 模板的脚手架工具

## 使用

### npx

```shell
$ npx @andylzc/sug-cli create <new app>
```

### 全局安装

```shell
$ npm i @andylzc/sug-cli -g
```

```shell
$ sug-cli create <new app>
```

## 本地开发调试

```shell
$ cd sug-cli && npm link
$ cd ..
$ sug-cli create <new app>
```
